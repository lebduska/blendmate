# Knowledge Base

This directory contains Blendmate knowledge artifacts organized by source and Blender version.

## Content policy

- Summaries in info.md are original, human-written descriptions.
- Facts are derived from public sources listed in each node's links.json.
- Do not copy text verbatim from external documentation.

## Source references

Common sources include:
- Blender Manual (https://docs.blender.org/manual/)
- Blender Python API (https://docs.blender.org/api/current/)
- Blender source repository (https://github.com/blender/blender)
